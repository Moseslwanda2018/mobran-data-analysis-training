# Mobran Data Analysis Training

Welcome to the official Certified Data Analysis Training in R repository by Mobran Data Solutions.

## 📊 What You Will Learn
- Data cleaning in R
- Exploratory data analysis
- Data visualization
- Building dashboards with Shiny

## 📁 Project Structure
- data/ → datasets used in training
- scripts/ → R scripts
- outputs/ → results and visualizations
- docs/ → training materials

## 🚀 Getting Started
1. Install R and RStudio
2. Clone this repository
3. Run scripts in the scripts/ folder
